# Changelog

All notable changes to the "Cyberpunk Accessible" theme will be documented in this file.

## [1.0.0] - 2025-11-04

### Added
- Initial release of Cyberpunk Accessible theme
- High-contrast color scheme optimized for color blindness
- Support for HTML, JavaScript, JSX, TypeScript, Python, CSS, and SCSS
- Cyberpunk aesthetic with neon colors (magenta, cyan, neon green, orange, hot pink)
- Maximum contrast between all syntax elements
- Semantic highlighting support
- Bold and italic font styles for additional differentiation
- Comprehensive syntax coverage for all supported languages

### Color Choices
- Avoided blue/purple and yellow/green combinations
- Used luminance contrast in addition to hue differences
- All foreground/background pairs meet WCAG AAA standards
- Distinct colors for tags, attributes, values, keywords, functions, classes, strings, and numbers

### Accessibility Features
- Comments are dimmed but still readable (gray + italic)
- Active line numbers highlighted in theme color
- Cyan cursor for high visibility
- Error highlighting with bold red
- Maximum contrast UI elements with magenta borders
